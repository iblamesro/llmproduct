# Data Directory

This directory contains documents for the RegIntel AI application.

## Structure

- `sample_documents/` - Place your regulatory documents here (PDF, DOCX, TXT)
- Uploaded documents are stored temporarily during processing

## Supported Formats

- PDF (.pdf)
- Word Documents (.docx)
- Text Files (.txt)
- Markdown (.md)

## Usage

1. Place regulatory documents in the `sample_documents/` folder
2. Use the "Load from Data Folder" option in the app
3. Or upload documents directly through the web interface
